# 全渠道电商零售 Demo：指标蓝图

> 本文是指标树建设前的设计清单。正式指标文件将在下一阶段按 Hs 文件契约生成。

## 1. 核心指标清单

### 经营结果与利润

| id         | 指标    | 定义或公式                      | 主要用途        |
| ---------- | ----- | -------------------------- | ----------- |
| `IND-0001` | GMV   | 支付成功订单的商品成交金额，退款前口径        | 经营规模        |
| `IND-0002` | 退款金额  | 统计期内确认退款的商品金额              | 从成交还原真实收入   |
| `IND-0003` | 净销售额  | `GMV - 退款金额`               | 核心销售结果      |
| `IND-0004` | 商品成本  | 已售商品对应的采购成本                | 毛利计算        |
| `IND-0005` | 毛利润   | `净销售额 - 商品成本`              | 商品经营质量      |
| `IND-0006` | 毛利率   | `毛利润 / 净销售额`               | 跨规模比较商品盈利质量 |
| `IND-0007` | 营销费用  | 付费获客与营销活动支出                | 获客成本和利润计算   |
| `IND-0008` | 履约费用  | 仓储、打包和配送费用                 | 履约效率和利润计算   |
| `IND-0009` | 渠道费用  | 第三方渠道佣金与技术服务费              | 渠道利润计算      |
| `IND-0010` | 贡献利润  | `毛利润 - 营销费用 - 履约费用 - 渠道费用` | 第一版最终经营目标   |
| `IND-0011` | 贡献利润率 | `贡献利润 / 净销售额`              | 跨区域、渠道和品类比较 |

| `IND-0029` | 客单价 | `GMV / 支付订单数` | 单订单价值 |
| `IND-0030` | 平均销售单价 | `GMV / 销售件数` | 商品价格结构 |
| `IND-0031` | 退款率 | `退款金额 / GMV` | 成交质量 |
| `IND-0032` | 准时履约率 | `准时完成订单数 / 已完成订单数` | 履约质量 |

## 2. 主要结构拆解

```text
净销售额
├── 按区域：北区 + 东区 + 南区
├── 按渠道：自营App + 自营小程序 + 第三方平台
├── 按一级品类：厨房用品 + 收纳用品 + 清洁用品 + 小家电
└── 按客户类型：新客户 + 老客户
```

结构项只有在互斥、完整、同口径时才允许加总。不同维度的拆解不能互相相加。

## 3. 主要公式拆解

```text
GMV = 支付订单数 × 客单价
GMV = 销售件数 × 平均销售单价
支付订单数 = 支付客户数 × 购买频次
支付客户数 = 新支付客户数 + 老支付客户数

支付客户数
= 访问用户数
 × 访问详情转化率
 × 详情加购转化率
 × 加购提交转化率
 × 提交支付转化率

有货率 = 有货SKU数 / 在售SKU数
退款率 = 退款金额 / GMV
贡献利润 = 净销售额 - 商品成本 - 营销费用 - 履约费用 - 渠道费用
```

## 4. 条件性拆解

- 新客获客成本只适用于能识别营销归因的新支付客户，不适用于自然流量客户。
- 付费投放产出比只适用于具有明确营销来源和费用的流量，不得用全渠道净销售额作分子。
- 渠道费用主要适用于第三方平台；自营渠道如有支付技术费，应使用单独口径。
- 准时履约率只适用于已进入履约并完成的订单，取消订单不进入分母。
- 区域汇总的转化率、毛利率、退款率和履约率必须从基础分子分母重算，不能直接平均城市比率。
- 两条 GMV 公式都应成立；如果回算不一致，优先判定数据源或统计粒度存在问题。

## 5. 第一版故意保留的待补项

- 搜索和推荐算法的独立曝光、点击指标暂不纳入核心清单。
- 库存周转天数暂不进入第一版，因为需要稳定的期初、期末库存价值口径。
- 用户生命周期价值暂不计算，因为 30 个月数据不足以覆盖完整生命周期。
- 严格因果意义上的营销 ROI 暂不计算，第一版只有可归因净销售额与费用形成的投放产出比。
- 供应商履约表现暂不单独建指标，第一版只通过库存与有货率体现结果。

待补项用于演示 `hs-graph` 如何扩展指标树，不得被分析过程自动补成确定关系。
