# Hs Public Demo Workspace

这是由 `hs-data-assistant` 公开真源初始化的独立演示工作区。它用于运行任务、录屏、截图和制作公开安全的交付物。

## 开始一次演示

1. 打开 `business_graphs/demo-omnichannel-retail/tasks/README.md`。
2. 选择一张任务卡，把“用户命题”交给当前 AI 环境中的 `hs-entry`。
3. 确认 Hs 先读取图谱，再展示施工图并等待确认。
4. 完成后，把结果与 `tasks/expected_answers/` 中的答案对照。

## 目录

- `.agents/skills/`：本次初始化时复制的 Hs Skills 快照。
- `business_graphs/`：虚构零售业务图谱、合成数据、任务和答案。
- `demo_projects/active/`：正在进行的演示项目。
- `demo_projects/completed/`：完成并准备归档的演示项目。
- `outputs/`：报告、表格、图表和反馈卡。
- `exports/`：准备分享的导出文件。
- `recordings/`：录屏素材。
- `temp/`：可随时删除的中间文件。

## 真源关系

本工作区是公开产品包的可重置副本，不是产品真源。需要回写的改进先由 `hs-feedback` 生成建议，再人工修改上游 GitHub 仓库。不要把本工作区直接覆盖回上游。

## 数据声明

本工作区包含完全虚构的合成数据，仅用于 Hs 功能演示，不代表任何真实企业。
